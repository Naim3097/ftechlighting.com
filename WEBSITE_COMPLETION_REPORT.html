UI/UX Concept & Theme

Theme: "Industrial Luxury"

The design philosophy centers on precision, darkness, and illumination. Reflecting the company's core business (lighting), the interface uses a "Dark Mode" default to allow content (light) to stand out.

  • Primary: Deep Black (#050505) & Burgundy (#820000)
  • Texture: Glassmorphism (Blur 10px) & Grain
  • Typography: Helvetica Neue (Clean, Swiss Style)

Core UX Principles

1. Discovery via Motion
Users "discover" content through interaction (hover reveals, scroll expansions). Nothing is static.
2. Cinematic Navigation
The site behaves less like a document and more like a film strip, moving horizontally to tell a linear story.

Desktop Scroll Behavior
Mechanism: `wheel` event listener transforms vertical scroll delta into horizontal translation.
Physics: Cubic-bezier(0.645, 0.045, 0.355, 1.000) for smooth, weighted momentum.
Snap: Sections snap to viewport width (100vw) to ensure focused viewing.
Mobile Scroll Behavior
Mechanism: Standard vertical scrolling (Native).
Adaptation: Horizontal layouts stack vertically. Complex hover effects are replaced by tap-to-reveal or static visibility.
Navigation: Hamburger menu becomes primary nav anchor.

Advanced Layout Kit

The FTECH identity relies on four distinct layout engines. Each handles content reveal differently.

Layout A: "The Curtain" Used on: Home Page
.accordion-panel {
  flex: 1;
  transition: flex 0.8s cubic-bezier(0.25, 1, 0.5, 1);
  border-right: 1px solid rgba(255, 255, 255, 0.05);
}
.accordion-panel:hover { flex: 3; }
Layout B: "The Strip" Used on: Services (Core Competencies)
.strip-item {
  flex: 1;
  transition: flex 0.8s cubic-bezier(0.25, 1, 0.5, 1);
}
.strip-item:hover { flex: 4; }
Layout C: "The Slice" Used on: Projects Hero
.slice-strip {
  transform: translateY(100%);
  transition: transform 1.2s cubic-bezier(0.19, 1, 0.22, 1);
}
.slice-strip:nth-child(odd) { transform: translateY(-100%); }
Layout D: "The Mask" Used on: Services Hero
h1 {
  mix-blend-mode: difference;
  letter-spacing: -8px;
  transition: letter-spacing 1.5s;
}
h1:hover { letter-spacing: -4px; }
